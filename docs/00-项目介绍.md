# DeskCalendar 项目介绍

> 版本：v1.0-draft ｜ 最后更新：2026-07-07

## 这是什么

**DeskCalendar** 是一个开源的 Windows 桌面日历应用，用于**替代点击系统托盘时钟时弹出的 Windows 原生日历面板**（即"托盘日历弹窗"），在观感与交互上对标 **360 小清新日历**。

核心体验：

- 点击任务栏托盘时钟图标 → 弹出日历面板（MVP 为方角不透明 + DWM 阴影；圆角 / 每像素透明 / 贴时钟定位见 ADR-03 后续补全）
- 显示公历网格 + 农历 / 节气 / 节假日 / 调休
- 弹窗不抢占焦点、毫秒级响应、关机无残留

未来扩展（路线图）：待办（Todo）、天气（Weather）、主题换肤、插件系统、自动更新。

## 愿景

成为 Windows 上最轻量、最可定制、完全开源的托盘日历：

- 零后台常驻服务
- 零强制联网（农历 / 节假日完全离线）
- 零 CGO 依赖（`CGO_ENABLED=0` 可编译）

## 目标（Goals）

| 编号 | 目标 | 验收 |
|------|------|------|
| G1 | 弹窗毫秒级、不卡顿、不抢焦点 | 点击到可见 < 50ms；焦点留在原窗口 |
| G2 | 无边框 + DWM 阴影，观感对齐 360（圆角 / 每像素透明降级，待后续补全） | 无边框 + DWM 阴影已落地（上游 gogpu 原生）；圆角/每像素 alpha 列为 ADR-03 后续补全 |
| G3 | 农历/节气/节假日/调休准确（离线） | lunar-go + holiday-cn 数据校验通过 |
| G4 | 单一二进制，免安装或极简安装 | 产出单个 exe；可选 NSIS 安装包 |
| G5 | 完全开源、可审计、可贡献 | MIT 许可；清晰 CONTRIBUTING |

## 非目标（Non-Goals）

- **不做壁纸模式**（WorkerW 桌面嵌入）——当前架构为托盘弹窗；壁纸模式列为未来探索（见 `20-Platform/WorkerW.md`）
- 不做完整 PIM / 邮件 / 通讯录
- 不做服务器端 / 多端同步（MVP）

## MVP 范围（v1.0）

- 托盘弹窗 + 无边框面板 + DWM 阴影（**降级形态**：圆角 / 每像素透明 / 贴时钟定位待 ADR-03 后续补全；原「圆角透明 POC」基于已 stash 的本地 patch gogpu，现已过期）
- 公历网格 + 农历 / 节气 + 节假日 / 调休（lunar-go + holiday-cn，离线）
- 基础主题（跟随系统浅 / 深）
- 最小设置面板（开机自启、主题、弹窗位置）

## 技术栈

| 层 | 选型 | 决策 |
|----|------|------|
| UI 框架 | `gogpu/ui`（Signal 响应式，wgpu GPU 渲染） | ADR-01 |
| 托盘 | `gogpu/systray` | ADR-02 |
| 窗口样式 | 无边框 + DWM 阴影（圆角 / 每像素透明待后续补全） | ADR-03 |
| 毛玻璃 | 跳过 Mica（自绘渐变替代） | ADR-04 |
| 数据 | lunar-go / holiday-cn / Open-Meteo | ADR-05 |
| 语言 | Go 1.25+，`CGO_ENABLED=0` | ADR-06 |
